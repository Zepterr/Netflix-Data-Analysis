## 📌 Overview
A complete Exploratory Data Analysis (EDA) of Netflix's content catalog 
using Python. This project uncovers patterns in content type, genre 
distribution, geographic production, ratings, and seasonal trends — 
concluding with a content-based recommendation system.

**Course:** Introduction to Data Science | Spring 2026  
**Dataset:** Netflix Titles — Kaggle (8,794 titles, 12 features)

## 📊 Key Findings
- Netflix is 69.7% Movies and 30.3% TV Shows
- Peak content growth was 2019 — COVID caused a sharp decline after
- USA dominates with 3,680 titles; India is 2nd with 1,046
- TV-MA is the most common rating — Netflix targets adult audiences
- July is the most active month for new content additions
- Bandersnatch is the longest title at 312 minutes

## 🤖 Recommendation System
Content-based recommender using TF-IDF vectorization and cosine 
similarity across genres, directors, cast, and descriptions.
